# 📦 Nama Project Laravel

Deskripsi singkat tentang project kamu. Jelaskan tujuan aplikasi ini dan fitur utamanya.

## 🚀 Requirements

Pastikan sudah terinstall:

- PHP >= 8.x
- Composer
- MySQL / PostgreSQL / SQLite
- Node.js & npm
- Git

## 📥 Cara Install

Clone repository:

```bash
git clone https://github.com/DionAhza/Payroll_app/
```

```bash
cd nama-project
```

```bash
composer install
```

```bash
copy .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan migrate
```

```bash
php artisan serve
```
